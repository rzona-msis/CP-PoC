Commitment to Accessibility

Campus Resource Hub is designed with accessibility at its core, meeting WCAG 2.1 Level AA standards.

  • Full keyboard navigation support
  • Screen reader compatibility
  • High contrast color schemes
  • Semantic HTML structure
  • ARIA labels and landmarks
  • Responsive design for all devices

If you experience any accessibility barriers, please contact us at accessibility@campusresourcehub.edu

Technology Stack

Built with modern, secure web technologies:

  • Backend: Python 3.10+ with Flask 3.0
  • Database: SQLite/PostgreSQL with SQLAlchemy ORM
  • Authentication: Flask-Login with bcrypt password hashing
  • Frontend: Bootstrap 5, semantic HTML5, accessible JavaScript
  • Security: CSRF protection, XSS prevention, SQL injection prevention

User Roles

Students

Browse resources, create bookings, send messages, and leave reviews to help the community.

Staff

Create and manage resources, approve booking requests, and engage with the campus community.

Administrators

Oversee platform operations, manage users, moderate content, and ensure smooth operation.
